# Twitter clone

Twitter clone is a simple system that lets users to

  - Create a tweet
  - Retweet
  - Delete tweet

# Setup

Clone repository, setup ruby 2.4.1 using RVM
```
bundle install && rake db:migrate
rails server
```
visit localhost:3000/
